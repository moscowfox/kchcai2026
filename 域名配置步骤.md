# 域名配置步骤（最简单方式）

## 📝 配置域名 atlantis9.com

### 步骤 1：登录腾讯云控制台
1. 访问 https://console.cloud.tencent.com/
2. 登录您的账号

### 步骤 2：进入 EdgeOne Pages
1. 在控制台搜索 "EdgeOne" 或 "边缘安全加速"
2. 进入 **EdgeOne** → **Pages**（静态网站托管）

### 步骤 3：找到您的部署
1. 在 Pages 列表中，找到刚才部署的网站
2. 点击进入详情页

### 步骤 4：绑定自定义域名
1. 在网站详情页，找到 **"自定义域名"** 或 **"域名绑定"** 选项
2. 点击 **"添加域名"** 或 **"绑定域名"**
3. 输入您的域名：`atlantis9.com` 和 `www.atlantis9.com`
4. 系统会提示您需要配置 DNS 记录

### 步骤 5：配置 DNS 记录
在您的域名注册商（如阿里云、腾讯云、GoDaddy 等）的 DNS 管理页面：

1. 添加 **CNAME 记录**：
   - **主机记录**：`@`（或留空，表示主域名）
   - **记录类型**：`CNAME`
   - **记录值**：EdgeOne 提供的 CNAME 地址（通常在绑定域名时会显示）
   - **TTL**：600（或默认值）

2. 添加 **CNAME 记录**（www 子域名）：
   - **主机记录**：`www`
   - **记录类型**：`CNAME`
   - **记录值**：与上面相同的 CNAME 地址
   - **TTL**：600

### 步骤 6：等待生效
- DNS 记录通常需要 **5-30 分钟** 生效
- 在 EdgeOne 控制台可以看到域名状态变为 **"已生效"** 或 **"已解析"**

### 步骤 7：验证访问
等待 DNS 生效后，访问：
- http://atlantis9.com
- http://www.atlantis9.com

## 💡 提示
- 如果找不到 EdgeOne Pages，可能需要在控制台左侧菜单中查找 **"边缘安全加速"** → **"静态网站"**
- 如果遇到问题，可以联系腾讯云客服或查看 EdgeOne Pages 的官方文档
- DNS 生效后，EdgeOne 会自动配置 HTTPS（SSL 证书）
