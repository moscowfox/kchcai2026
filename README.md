# 北京康城合创官方网站

这是北京康城合创科技有限公司的官方网站项目。

## 项目结构

```
kchcaiweb/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # JavaScript 交互逻辑
└── README.md       # 项目说明文档
```

## 功能特性

- 🎨 现代化深色主题设计
- 📱 完全响应式布局，支持移动端
- ✨ 流畅的滚动动画效果
- 🎯 平滑的页面导航
- 📞 联系方式一键拨打/发送邮件
- 🚀 优化的性能和加载速度

## 技术栈

- HTML5
- CSS3 (使用 CSS 变量、Grid、Flexbox)
- 原生 JavaScript (无依赖)
- Intersection Observer API (滚动动画)

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可查看网站
2. 或者使用本地服务器运行（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 主要板块

- **Hero 首屏**：品牌展示和主要行动号召
- **核心优势**：公司三大核心优势展示
- **业务版图**：AI算力与云基座、智能识别与安防
- **服务体系**：咨询规划、系统集成、运维支持

- **发展历程**：公司发展时间线
- **联系我们**：联系方式信息

## 自定义配置

可以通过修改 `styles.css` 中的 CSS 变量来自定义主题颜色：

```css
:root {
    --bg-color: #050a14;
    --accent-cyan: #00f2ff;
    --accent-purple: #bc13fe;
    /* ... 其他变量 */
}
```
